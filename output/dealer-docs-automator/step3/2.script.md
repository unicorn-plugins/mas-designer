## 슬라이드 5: 현상 — 3대 고통
**패턴**: E (카드 그리드 3개)
**의도**: 정량 수치로 현실 고통 제시

### 콘텐츠 카드
1. **반려율 15~25%** — 본사 BSS 재처리 연 150~300억 비용
2. **사기 개통 0.3~1.2%** — 건당 100~300만 원 손실 + 법적 리스크
3. **처리 시간 25~40분** — 영업사원 소진, 경쟁 열위

### 발표자 노트
> 이 3대 지표는 업계 통상치이며, 우리 사의 실측값도 이 범위 안에 있습니다.

## 슬라이드 6: 근본 원인
**패턴**: D (테이블)
**의도**: 5WHY 근본 원인 3종 요약

### 콘텐츠 (테이블 3행)
| 현상 | 근본 원인 |
|------|----------|
| 반려율 20% | 규정이 현장 작성 시점에 실시간·맥락적으로 제공 안 됨 |
| 사기 0.8% | 위험 신호가 개통 순간 현장에 도달하지 않음 (킹핀) |
| 30분 처리 | 자동화 가능 작업이 오케스트레이션 부재로 순차 수작업 |

### 발표자 노트
> 세 가지 모두 "데이터와 지능이 있으나 현장 순간에 도달하지 않는 구조적 격차" 문제입니다.

## 슬라이드 9: MAS 아키텍처 ① — 에이전트 구성
**패턴**: D (테이블)
**의도**: 7개 전문 에이전트 프로파일

### 콘텐츠 (테이블 7행)
| # | 에이전트 | Tier | 핵심 역할 |
|:-:|---------|:----:|----------|
| 1 | Orchestrator | HIGH | State 관리·병렬 호출·HITL 분기 |
| 2 | ID Verifier (IDV+) | MEDIUM | OCR+VLM+얼굴+라이브니스 |
| 3 | Compliance RegGuard | MEDIUM | 규정 RAG + ReAct 조항 인용 |
| 4 | Fraud Scorer | HIGH | ML+LLM 앙상블 사기 스코어 |
| 5 | Doc Generator | LOW | PDF 3종 자동 생성 + 전자서명 |
| 6 | Audit Logger | LOW | Append-only 감사 로그 |
| 7 | Dealer Copilot | MEDIUM | 영업사원 질의응답 (선택) |

### 발표자 노트
> 각 에이전트는 단일 전문성에만 집중하며, 호출은 Orchestrator가 전담해 월권을 차단합니다.

## 슬라이드 10: MAS 아키텍처 ② — LangGraph 플로우
**패턴**: C (플로우)
**의도**: 병렬 오케스트레이션 시각화

### 콘텐츠 (플로우 6단계)
1. **INTAKE** → 이벤트 수신·PII 마스킹
2. **IDV ∥ RegGuard ∥ Fraud** → 2초 병렬 평가
3. **JOIN / VERDICT** → Green/Yellow/Red 판정
4. **ESCALATE** (YELLOW/RED) → HITL
5. **DOC_GEN / SIGN** → 서류·서명
6. **BSS_SUBMIT / LOG** → 제출·감사

### 시각 요소
- **이미지**: `![LangGraph 아키텍처](images/slide-10-architecture.png)` (중앙)

### 발표자 노트
> 핵심은 2초 병렬 + HITL입니다. BSS 장애 시 Local Queue로 업무 연속성 보장합니다.

## 슬라이드 12: 신뢰성·보안
**패턴**: D (테이블)
**의도**: CIO 거버넌스 관심사 응답

### 콘텐츠 (테이블 5행)
| 항목 | 설계 |
|------|------|
| SLA | 병렬 Hard 2.5s, End-to-End 30min |
| Circuit Breaker | MCP별 1분 실패율 >50% → OPEN |
| HITL | Yellow=점장 / Red=리스크팀 승인 필수 |
| PII | 로컬 암호화 + 클라우드 전송 전 마스킹 |
| 감사 | Append-only, SHA-256 체인, 7년 보관 |

### 발표자 노트
> 100% 자동 판정은 금지하며, KYC/AML 규제 요구를 그대로 반영했습니다.

## 슬라이드 14: 기대 효과
**패턴**: E (카드 그리드 4개)
**의도**: 수치 목표 제시

### 콘텐츠 카드
1. **반려율 20% → 5%** (-75%) · BSS 재처리 연 150~300억↓

### 발표자 노트
> 보수 시나리오 기준이며 공격적 확산 시 12개월 회수도 가능합니다.
