# 비즈니스 가치

> 근본원인 R1~R3를 해결했을 때 얻는 고객·기업 가치.
> problem-analyst 산출 — 정량(추정)·정성 분리.

---

## 고객 가치 (영업사원·점장 중심, 2차로 최종 고객)

### 영업사원 (Primary)

| 가치 | 정량 추정 | 정성 |
|------|----------|------|
| 서류 작성 시간 단축 | 건당 **25~40분 → 8~12분** (약 -60%) | "본사에 안 물어봐도 95% 처리됨"의 자신감 |
| 반려 재처리 감소 | 반려율 15~25% → **3~5%** | 재방문 안내·사과 스트레스 제거 |
| 사기 의심 판단 지원 | 현장 위험 스코어 **2초 내 표시** | "내가 이 고객 받아도 되나" 불안 감소 |
| 처리량 증가 | 일 5~8건 → **9~13건** | 실적·인센티브 향상 |
| 교육 부담 완화 | 신입 숙련기간 3개월 → **4~6주** | 규정 학습 부담 감소, 이직 유인 완화 |

### 대리점 점장 / 고객

- **점장**: 일 단위 반려·리스크 대시보드 제공, 사고 예방으로 본사 감사 대응 부담 감소
- **최종 가입자**: 개통 소요 **40분 → 15분**, 당일 개통률 상승, 불완전판매 방지로 철회·민원 감소

## 기업 가치 (통신사 본사·위탁대리점 경영)

### 본사 (MNO)

| 가치 영역 | 정량 추정 (대리점 1,000점 기준 연) | 정성 |
|---------|--------------------------------|------|
| **운영비 절감** | BSS 재처리 비용 **-150~300억 원/년** (반려 재제출 감소) | 영업지원팀 콜볼륨 -40~60% |
| **사기 손실 감소** | 개통 사기 손실 연 **-50~120억 원** | 브랜드 리스크 감소, 언론 노출 방지 |
| **과태료·규제 리스크** | 개인정보 유출·KYC 미흡 과태료 **-5~30억 원** (사건 당 1~10억) | 감사·컴플라이언스 심의 통과율 향상 |
| **매출 증대** | 영업사원 처리량 +50% → 개통 매출 **+5~8%** | 경쟁 대리점 대비 우위 |
| **데이터 자산** | 사기·반려·불완전판매 패턴 축적 | 전체 CX·CRM 전략에 재활용 |

### 위탁 대리점 (직접 수익자)

- **수수료 수익 +5~8%** (개통 건수 증가에 비례)
- **영업사원 이직률 -10~15%** (업무 부담 완화)
- **본사 계약 유지·확대** (KPI 달성으로 대형 대리점 승격)

### 리스크 관점 가치

| 리스크 | 현재 노출 | MAS 도입 후 |
|-------|----------|------------|
| 개인정보 유출 | 신분증 수기·스캔·이메일 전송 | 로컬 OCR + PII 마스킹 후 전송, 감사 로그 |
| 명의도용 법적 책임 | 대리점/본사 공동 책임 논란 | 실시간 위험 스코어·차단 로그로 방어 가능 |
| AML/KYC 감독 강화 대응 | 수동 체크리스트 | 에이전트 자동 KYC 스코어링·보고 |

## 3대 핵심 수치 목표 (KPI)

> 경영진 보고(CIO)에 강조할 지표.

| KPI | 현재 | 목표(12개월) | 산정 근거 |
|-----|:----:|:-----------:|----------|
| 개통 서류 반려율 | 20% | **≤5%** | 규정 실시간 검증 + 자동 서식 → 오류 95% 차단 |
| 사기 개통 발생률 | 0.8% | **≤0.1%** | 위조·도용·패턴 실시간 탐지로 90% 이상 사전 차단 |
| 건당 서류 처리 시간 | 30분 | **≤12분** | 병렬 에이전트 + 자동 서식 + 본사 문의 대체 |

## ROI 개요 (요약)

- **도입 비용(파일럿→전사)**: 30~60억 원 (1년차 기준, 인프라 + LLM API + 구축)
- **연간 절감 + 증대 효과**: 200~450억 원 (반려 감소 + 사기 손실 감소 + 매출 증대)
- **ROI 회수 기간**: **약 15~22개월** (보수 시나리오), 공격적 확산 시 12개월 내
