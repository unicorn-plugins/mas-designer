## 후보 1. 실시간 멀티에이전트 리스크 트리아지 (RTA)

**포함 발산 항목**: S2, S3, S7, S8, S13, S15, S20, S25

**설명**: 개통 접수 순간 **OCR · KYC · 약정이력 · 경찰청 DB · 이상 패턴**을 병렬 에이전트로 평가하여
**Green/Yellow/Red 3단계 위험 스코어**를 2초 내 제공. LangGraph 오케스트레이션 + 본사 BSS 전 단계 차단.

**핵심 AI 기술**:
- LangGraph (에이전트 병렬·조건부 라우팅)
- 실시간 ML 스코어링 (Gradient Boosting + LLM 추론 앙상블)
- MCP 기반 외부 DB 연동 (KYC, 경찰청 분실신고, 본사 블랙리스트)

**차별성**: 킹핀 직접 해결. 카드사 FDS 수준의 실시간성을 통신에 이식.

---

## 후보 2. 멀티모달 신원확인 & 위변조 탐지 (IDV+)

**포함 발산 항목**: S1, S16, S19, S21, S22

**설명**: 태블릿/PC 카메라로 **신분증 다각도 촬영 + 고객 셀피 + 라이브니스 체크**.
VLM 기반 위변조 특징 탐지(합성·변조·재촬영) + 얼굴 대조로 명의도용 사전 차단.

**핵심 AI 기술**:
- Vision Language Model (Gemini 2.5 Pro / GPT-4o Vision / Claude Sonnet Vision)
- Face embedding (ArcFace 등) + 라이브니스 (active challenge)
- OCR + NER (PII 즉시 마스킹)

**차별성**: IDV 전문업체(Onfido/Jumio) 수준 기능을 통신사 자체 구축 또는 SaaS 통합.

## 후보 3. 규정 RAG 컴플라이언스 게이트웨이 (RegGuard)

**포함 발산 항목**: S4, S11, S17, S18, S23, S27

**설명**: 본사 규정집·상품 카탈로그·KYC/AML 규제 문서를 **Vector DB 인덱싱** + 영업사원 질의·자동 검증.
ReAct 에이전트가 **조항 인용 + 판정 근거** 제시.

**핵심 AI 기술**:
- RAG (청킹·리랭킹·하이브리드 검색)
- ReAct 프롬프트 (근거 수집 → 결론)
- MCP 기반 규정 문서 서버

**차별성**: 리걸테크 인용 패턴을 통신 규정에 이식. R1 근본원인 자연 흡수.

---

## 후보 4. 서류 자동 생성 & 전자서명 통합 (AutoDoc)

**포함 발산 항목**: S12, S24

**설명**: 검증 완료 정보로 **위임장·가입신청서·동의서**를 템플릿 기반 자동 생성.
태블릿 **전자서명 패드** 연계로 출력·스캔 제거. 본사 BSS에 PDF+메타데이터 동시 제출.

**핵심 AI 기술**:
- 템플릿 엔진 + LLM 문구 보정
- 전자서명 API (공인·사설)
- PDF 생성·검증

**차별성**: R3 근본원인(수기 처리) 직접 완화. 선행 에이전트 출력을 그대로 자동 기입.

## 후보 5. 영업사원 코파일럿 (DealerCopilot)

**포함 발산 항목**: S18, S29, S26

**설명**: 영업사원 옆에서 **고객 응대 스크립트 · 약관 설명 · 철회 예측 · 본사 Q&A**를 실시간 제공.
Human-in-the-Loop 보조, 결정권은 영업사원.

**핵심 AI 기술**:
- 대화형 LLM (긴 컨텍스트 + 메모리)
- RAG (상품·혜택·경쟁사 정보)
- 예측 모델 (철회 확률)

**차별성**: 증강(Augmentation) 대표 케이스. CDSS-영감 설계.

---

## 후보 6. 통합 감사·대시보드 인텔리전스 (AuditSuite)

**포함 발산 항목**: S10, S28

**설명**: 모든 에이전트 판정·사용자 확인·본사 제출을 **구조화 로그 + 자동 요약**.
점장용 일일 이상 징후 대시보드, 본사 컴플라이언스 월간 리포트 자동 생성.

**핵심 AI 기술**:
- 이벤트 스트리밍 + Vector DB 저장
- LLM 요약·이상 탐지
- Chart.js/Dashboard 자동 생성

**차별성**: 단일 실행을 다목적 활용. 사후에도 데이터 자산화 가능.
